n8n Form → Google Sheets → Gmail (10 Steps)

Open n8n and click New Workflow

Add trigger On form submission

Create and configure the form fields (name, email, message, etc.)

Add Google Sheets – Append row in sheet

Connect On form submission → Append row in sheet

Select your Google Sheet and map form fields to columns

Add trigger Google Sheets Trigger

Set trigger event to Row Added

Add Gmail – Send a message node

Connect Google Sheets Trigger → Gmail, map row data to email, then Activate

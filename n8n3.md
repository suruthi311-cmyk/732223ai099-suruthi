Open n8n and click New Workflow

Add trigger On form submission

Configure the form fields (name, email, message, etc.)

Add node AI Agent

Connect On form submission → AI Agent

Click Chat Model* on the AI Agent

Select Google Gemini Chat Model and add credentials

Add node Gmail – Send a message

Connect AI Agent → Gmail and map the AI response to email body

Execute workflow, submit the form, then Activate the workflow

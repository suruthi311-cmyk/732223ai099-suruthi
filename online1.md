Open n8n and click New Workflow

Click Add node → select Webhook

Set HTTP Method to GET

Set Path to click-email

Save the Webhook node

Copy the Test Webhook URL

Click Add node → select Email (or Gmail/SMTP)

Connect Webhook → Email

Set To, Subject, and Message in the Email node

Execute workflow, click the link, then Activate the workflow

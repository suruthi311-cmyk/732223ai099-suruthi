n8n Chat → AI Agent 

Open n8n and click New Workflow

Add trigger When chat message received

Click Add node → choose AI Agent

Connect When chat message received → AI Agent

Click the Chat Model* input on the AI Agent

Select Google Gemini Chat Model

Add / select your Google Gemini API credentials

Choose the Gemini model (example: gemini-pro)

In AI Agent settings, set instructions (optional system prompt)

Execute workflow and send a chat message to test

      Docker Installation 

Open browser and go to docker.com
Navigate to Docker Desktop download.

Select your operating system
Download the installer file.

Run the Docker installer
Start the installation process.

Accept license and permissions
Allow system access if prompted.

Follow on-screen setup steps
Keep default configuration.

Complete installation process
Restart system if required.

Open Docker Desktop
Wait until Docker starts.

Check Docker status
Ensure it shows Running.

Open Terminal or Command Prompt
Run docker --version.

Installation is successful
Docker is ready to use

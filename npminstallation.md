          npm Installation 

Open a web browser and visit nodejs.org
Download the LTS version for your OS.

Run the downloaded installer file
Accept the license agreement.

Click Next through the setup wizard
Keep default options selected.

Ensure npm is checked in features
It installs automatically with Node.js.

Click Install to start installation
Wait until the process completes.

Finish the installation and close setup
Restart system if prompted.

Open Terminal / Command Prompt
Use administrator access if needed.

Run node -v command
Confirms Node.js is installed.

Run npm -v command
Confirms npm is installed.

Installation is complete
npm is ready to use
